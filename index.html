Introduction

Python is a high level programming language with an emphasis in code readability. Mostly used in back-end, its object oriented approach aims to help programmers write clear, logical code for small and large-scale projects.

It has a dynamic typing, which helps new Python Devs to write code without paying attention to the variable's type. Also, it has strong typing, so any operation between variables with different types will lead to an error.

Why learn Python?

Because its readability, strong typing, dynamic typing and simple syntax allows new learner to better understand it, unlike other languages like C or Java. The algorithm implementation in Python is easier due to the little attention a student has to pay in order to write code.

Initializing and Declaring Variables

Unlike other programming languages, variables are declared and initialized using the assignment operator " = ":

myHouse = 'Slytherin'

If you try to use an uninitialized variable, the code will print an error. That is why you always have to initialize it first.

Indentation

In Python, we don't use bracket notation synthax to wrap a statement's block. Instead, what gives us the guideline for any statement's block is the Indentation. That is, the space placed left to a code line. Because of that we can know which elements belongs to the parent, or which elements are childs of its parent element.
